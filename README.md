# U.S. NAICS Industry Revenue Analysis Portfolio

A comprehensive 5-project data analysis portfolio exploring U.S. industry opportunities through NAICS codes and geographic regions. Built while learning SQL and Python from scratch as a beginner data analyst.

## 📊 The 5 Projects

**Project 1: Identifying Themes**  
High-level analysis of top NAICS industries by state and census region using Census Bureau data.

**Project 2: Discovering Connections**  
Analysis of economic drivers (population, income, etc.) that influence industry revenue across states.

**Project 3: Categorizing Things**  
Segmentation of state-industry markets into High Growth, Stable, Declining, and Emerging categories.

**Project 4: Spotting Something Unusual**  
Outlier detection highlighting surprising industry opportunities and risks by state with interactive dashboard.

**Project 5: Making Predictions**  
Time-series forecasting of NAICS revenue opportunities for 2026–2028.

## 🛠️ Technical Skills Demonstrated
- Python (pandas, matplotlib, seaborn, Prophet)
- SQL (SQLite queries for data exploration)
- Data cleaning and transformation
- Statistical analysis (Z-score, correlations)
- Interactive dashboards (Looker Studio)
- Time-series forecasting
- Data storytelling and visualization

## 📁 Repository Structure
- `/Project-1` to `/Project-5`: Individual project notebooks and analysis
- `/data`: Cleaned datasets from U.S. Census Bureau
- `/dashboards`: Looker Studio dashboard links
- `/docs`: Written insights and case studies

## 🎯 Learning Journey
This portfolio was created following a structured 12-week "Weekend Warrior" plan (6–8 hours per week) while learning SQL and Python. Each project builds progressively on the previous one, showcasing both technical growth and the ability to deliver actionable business insights.
