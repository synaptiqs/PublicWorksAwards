# Active U.S. Public Construction Projects Analysis

**Technical Data Analysis Portfolio** exploring active federal construction contracts using USASpending.gov data (March 2026).

Built while learning Python and SQL, combining real-world construction domain expertise with data analytics. Focuses on infrastructure, heavy civil, and building construction projects.

---

## 📊 The 5 Projects

- **Project 1**: Geospatial and categorical analysis of active construction awards
- **Project 2**: Statistical analysis of cost drivers, regional trends, and economic factors
- **Project 3**: RFM-style segmentation of active projects (Value, Risk, Performance tiers)
- **Project 4**: Outlier detection and anomaly analysis using Z-score and deviation metrics
- **Project 5**: Time-series forecasting of construction pipeline using Prophet

---

## 🛠️ Tech Stack

- **Python 3.9+**
- **pandas** – Data manipulation and cleaning
- **NumPy** – Numerical computations
- **matplotlib + seaborn** – Static visualizations
- **Plotly** – Interactive visualizations
- **Prophet** – Time series forecasting
- **SQL** – Data querying (SQLite)
- **Jupyter Notebooks** – Analysis environment
- **Looker Studio** – Interactive dashboards

## 🚀 Installation & Setup

### 1. Clone the repository
```bash
git clone https://github.com/synaptiqs/NAICS_Portfolio.git
cd NAICS_Portfolio
```

### 2. Create virtual environment
```bash
python -m venv venv
source venv/bin/activate    # Windows: venv\Scripts\activate
```

### 3. Install dependencies
```bash
pip install -r requirements.txt
```

### 4. Download latest data
- Visit [USASpending.gov Download Center](https://www.usaspending.gov/download_center)
- Filter by NAICS codes: `236*`, `237*`, `238*`
- Download "Award" data (CSV format) and place in `/data/raw/`

### 5. Launch Jupyter
```bash
jupyter notebook
```
