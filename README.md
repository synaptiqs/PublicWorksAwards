# Public Works Awards
Active U.S. Public Construction Projects Analysis Portfolio

Comprehensive 5-project data analysis of ongoing federal construction contracts using USASpending.gov data (March 2026). 

Focuses on active infrastructure, highway, bridge, and building projects. Built while learning Python and SQL as a beginner data analyst with real-world construction industry experience.

Includes data cleaning, trend analysis, project segmentation, risk/opportunity detection, and pipeline forecasting.

Skills: Python (pandas, matplotlib, Prophet), SQL, Looker Studio

Perfect for showcasing both domain expertise and technical skills in public infrastructure data.
